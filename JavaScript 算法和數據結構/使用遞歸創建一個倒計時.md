### 題目

已經定義了一個函數 countdown，函數有一個參數（n）。 函數應該基於參數 n 遞歸調用返回 n 到 1 的連續數字的數組。 

如果函數以小於 1 的參數調用，函數應該返回空數組。 比如，用 n = 5 調用函數應該返回數組 [5, 4, 3, 2, 1]。 

函數必需使用遞歸函數調用自身，不能使用任何形式的循環。

### 解法

```js

// 只修改這一行下面的代碼

function countdown(n) {
  if (n <= 0) return []
  let arr = countdown(n - 1)
  arr.unshift(n)
  return arr
}
// 只修改這一行上面的代碼

```
